# 📊 財務KPI分析ダッシュボード

個人事業主・フリーランス向けの財務分析Webアプリ。
試算表のCSVまたはPDFをアップロードするだけで、KPI分析・資金繰り予測・節税試算まで自動化します。

## 主な機能

| 機能 | 概要 |
|------|------|
| 📈 KPI分析 | 試算表から収益性・安全性・効率性KPIを自動計算・可視化 |
| 🤖 PDF読み取り | AI（Claude）が試算表PDFを解析して自動入力 |
| 🎯 予実分析 | 予算vs実績の差異・達成率をゲージ・グラフで表示 |
| 📅 月次推移 | 複数月のKPI推移を折れ線グラフ＋移動平均＋異常値検知 |
| 🔮 着地見込み | 月次実績から年度末着地を予測・予算達成率を可視化 |
| 💰 資金繰り | 売掛金・支払予定から3ヶ月先の資金残高を予測、黒字倒産リスクスコアつき |
| 📉 損益分岐点 | 固定費・変動費率から損益分岐点売上高と感度分析を計算 |
| 💼 フリーランス試算 | 年収から手取り・所得税・住民税・社会保険を自動計算。iDeCo・共済の節税効果も試算 |
| 📊 前期比較 | 今期・前期のCSVを読み込んで全KPIの増減・成長率を自動比較 |
| 🏆 健全度スコア | 財務の健全性を収益性・安全性・効率性の3軸で0〜100点採点 |
| 🔬 What-if | 売上・コスト変化率のスライダーでKPIへの影響をリアルタイム確認 |
| 🏢 法人化判断 | 個人事業主と法人化の手取り・税負担を比較、最適役員報酬を可視化 |

## デモ

> デプロイ後にStreamlit CloudのURLを記載予定

## 技術スタック

- **フロントエンド/フレームワーク**: [Streamlit](https://streamlit.io/)
- **データ処理**: pandas
- **グラフ描画**: Plotly
- **PDF解析**: PyMuPDF (fitz)
- **AI連携**: [Anthropic Claude API](https://www.anthropic.com/)

## ローカルでの起動方法

```bash
# リポジトリをクローン
git clone https://github.com/<your-username>/financial-dashboard.git
cd financial-dashboard

# 依存関係をインストール
pip install -r requirements.txt

# APIキーを設定
cp .env.example .env
# .env を編集して ANTHROPIC_API_KEY を設定

# 起動
streamlit run app.py
```

## 環境変数

`.env` ファイルに以下を設定してください。

```
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxx
```
