# CFO Cockpit

> 経営者の「眠れない夜」を解消する資金繰りダッシュボード
> 年商1〜5億円の中小企業オーナー向け

freee や マネーフォワードのような会計ソフトは「過去の記録」までは提供しますが、
**「で、来月の資金は足りるのか？」「いつショートするのか？」「何をすべきか？」**には届きません。

CFO Cockpit はその**ラストワンマイル**——意思決定と行動指針——を埋めます。

## デモ

[https://financial-dashboard-2zpfnq7jneyn8pyb5xya55.streamlit.app/](https://financial-dashboard-2zpfnq7jneyn8pyb5xya55.streamlit.app/)

## 解決する課題

| 経営者の問い | このアプリが返す答え |
|------------|------------------|
| いつショートする？ | 日次の残高シミュレーションで、ショート日と残高をピンポイント表示 |
| なぜそうなる？ | 主要因の支払いを自動特定し、内訳を提示 |
| 最悪のケースは？ | 入金遅延・想定外支出のスライダーでシナリオ分析 |
| 何をすべきか？ | ルールベースのCFOアドバイザーが具体的な打ち手を提案 |

## 主な機能

### 💰 資金繰り3ヶ月先見（メイン機能）
- **日次残高シミュレーション**: 売掛金・買掛金・固定費から先90日の現金残高を予測
- **ショートポイント自動検出**: マイナスに沈む日を即座にアラート表示
- **シナリオ分析**: 「入金が30日遅れたら」「想定外支出500万が出たら」をスライダーで試算
- **リスクランク（A〜E）**: 月商比で資金繰りの健全性を5段階評価
- **ルールベースCFOアドバイス**: ショート対応・取引先集中リスク・固定費過重を自動診断
- **CSVインポート/エクスポート**: 入力データをCSV保存・次回読み込みで継続利用

### 📊 KPI診断
- **PDF AI読み取り**: freee/マネーフォワードの試算表PDFをClaude APIが自動解析
- **業種別ベンチマーク**: IT・コンサル・卸売・製造・建設・サービス業の6業種で比較
- **健全度スコア**: 収益性・安全性を100点満点で採点
- **AIコメント生成**: Claude APIがCFO目線で所見と打ち手を自動執筆

### 🔬 What-if シミュレーション
- 売上・原価率・固定費の変動が営業利益にどう影響するかをリアルタイム計算
- 損益分岐点ラインつき感度分析グラフ

## 設計思想

**「会計ソフト＋税理士＋銀行」では届かない領域を埋める。**

```
freee/MF       : [仕訳入力 → 試算表 → 申告データ]   ←「過去の記録」まで
税理士         : [制度対応 → 決算 → 申告]            ←「制度対応」まで
銀行           : [融資判断 → 返済管理]               ←「与信判断」まで

[経営判断 ←─── ここがラストワンマイル ───→ 行動]    ← CFO Cockpit
```

## 技術スタック

- **フレームワーク**: [Streamlit](https://streamlit.io/)
- **データ処理**: pandas
- **可視化**: Plotly（ダークテーマカスタム）
- **PDF解析**: PyMuPDF
- **AI**: [Anthropic Claude API](https://www.anthropic.com/)（claude-sonnet-4-6）

## ローカル起動

```bash
git clone https://github.com/<your-username>/financial-dashboard.git
cd financial-dashboard

pip install -r requirements.txt

# .env を作成し ANTHROPIC_API_KEY を設定
echo "ANTHROPIC_API_KEY=sk-ant-xxxx" > .env

streamlit run app.py
```
